Panibratets Artwear – E-commerce платформа для персоналізованого одягу

Інтерактивна вебплатформа для бренду «Panibratets Artwear», що дозволяє користувачам створювати унікальні дизайни одягу за допомогою 2D-конструктора та генеративного штучного інтелекту.

 Основні можливості

1. Магазин – каталог товарів з фільтрацією за розміром, ціною, матеріалом та категорією.
2. 2D-конструктор – створення персоналізованих макетів одягу (передня/задня сторона) з можливістю додавання тексту, фігур та зображень.
3. ШІ-асистент (AI Artist) – генерація унікальних ескізів за текстовими промптами через Hugging Face API (модель FLUX.1-schnell).
4. Блог– публікації з категоріями, тегами та підпискою на дайджест.
5. Особистий кабінет – збереження дизайнів, адрес доставки, історія замовлень.
6. Кошик та оформлення замовлення – підтримка оплати при отриманні та карткою.
7. Адмін-панель (CMS) – керування товарами, замовленнями, цінами, статтями блогу та перегляд аналітики.

Технологічний стек

Backend
1. .NET 9 + ASP.NET Core Web API
2. Entity Framework Core (Code-First, міграції)
3. SQLite (реляційна база даних)
4. JWT-автентифікація
5. Hugging Face Inference API (переклад + генерація зображень)

 Frontend
1. Angular 19 (Standalone Components, Signals API)
2. TypeScript
3. Fabric.js (2D-редактор)
4. Tailwind CSS v4 (стилізація)

 Запуск проекту

 Передумови
1 [.NET 9 SDK](https://dotnet.microsoft.com/en-us/download)
2. [Node.js](https://nodejs.org/) (v18 або вище)
3. [Angular CLI](https://angular.io/cli)

 Інструкція

1. Клонування репозиторію

2. Запуск бекенду (.NET)
   cd diploma
   dotnet restore
   dotnet run
   Сервер запуститься за адресою: https://localhost:7001 та http://localhost:5000

3. Запуск фронтенду (Angular)
   cd diploma.client
   npm install
   ng serve
   Додаток буде доступний за адресою: http://localhost:4200

4. Реєстрація першого адміністратора
   - Після запуску системи зареєструйте нового користувача
   - У базі даних вручну призначте йому роль Admin (або через Seed)
